### 간편 Git 사용 방법
  - Git clone
```
git clone https://github.com/yeonhojung/python.git
```
  - Git 다운받기
```
git pull
```
  - Git 올리기
```
git add .
git commit -am "message"
git push 
```

  - [Git 경고 메세지] LF will be replaced by CRLF in 해결 방안
> Windows, DOS 명령어
```
git config --global core.autocrlf true
```


> Linux, MAC 명령어
```
git config --global core.autocrlf input
```
